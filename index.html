ℹ️ How This Works

🟢 Live data — NHL Official API (free, no key required)
1
Data source
Stats are pulled automatically from the official NHL API (api-web.nhle.com). No manual entry needed.
2
Auto-updates every 3 hours
A GitHub Action runs fetch_stats.js automatically and commits a fresh data.json to the repo.
3
Scoring rules
Each playoff point (goal or assist) = 1 pool point. OT goals get a +1 bonus (so an OT goal = 2 pool pts total).
4
Manual refresh
Hit the ↻ Refresh button anytime to reload the latest stats.
